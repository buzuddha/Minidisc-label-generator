# Minidisc-label-generator
Minidisc label and slip case labels - makes ready to print sheets on US letter paper (8.5" x 11"). [You can purchase inexpensive label sticker paper here](https://www.onlinelabels.com/products/ol177?reward=oX7LjUfHj0w7M7RV7p7vEA%3d%3d)

# Input
This label maker allows you to search for albums in the MusicBrainz database. In the search bar. You can add an album via "+ add album" button. It will add the album MBID to the list. You can manually add new MBIDs by line to the list. The generator accomodates 14 lables per run. I frequently pack 2 albums on MDs when possible, so to accomodate this, there is an "+ add B side" button, which you can click after you've added the A side MBID.

# Output
The label make prints 14 labels & spine labels on page 1.
The label maker also prints slip case labels on pages 2 and 3.
